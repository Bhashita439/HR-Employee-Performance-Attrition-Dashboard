# HR-Employee-Performance-Attrition-Dashboard
An end-to-end HR analytics project using MySQL for data cleaning/analysis and Power BI for visual reporting and a business report 

---

# 🧠 Key Business Insights

The analysis helped identify important workforce patterns:

- Employee retention levels indicate differences across departments and employee groups.
- Workforce concentration is higher in specific locations and departments.
- Employee tenure analysis provides insights into retention behavior.
- Certain employee segments show higher turnover patterns, requiring further HR investigation.
- Workforce composition changes over time highlight organizational growth trends.
